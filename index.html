Markup Tags Table

Markup Tag Description Possible Attributes Description of Attributes
<abbr> Defines an abbreviation or an acronym dfn, title dfn: tag stands for the "definition element", and it specifies a term that is going to be defined within the content.
title: it must contain a full human-readable description or expansion of the abbreviation.
<b> Defines bold text sub, sup sub: tag defines subscript text. Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font.
sup: Superscript text appears half a character above the normal line, and is sometimes rendered in a smaller font.
<body> Defines the document's body background, text, link background: This attribute sets the background color or image of the web page.
text: This attribute sets the default text color for the web page.
link: This attribute sets the default color of hyperlinks that have not been visited.
<br> Defines a single line break global and event attributes global: can applied to nearly all HTML elements.
event: does not have specific event attributes, you can use global event attributes on it.
<div> Defines a section in a document id, class, style id: attribute is used to point to a specific style declaration in a style sheet.
class: specifies one or more classnames for an element.
style: attribute specifies an inline style for an element. The style attribute will override any style set globally, e.g. styles specified in the style tag or in an external sheet.
<footer> Defines a footer for a document or section don't have any special kind of attributes list, but it supports all global attributes like accesskey, data, class, contenteditable, dir, draggable, hidden, id, lang, dropzone, spellcheck, style, title, translate, etc. represents a footer for its nearest ancestor sectioning content or sectioning root element.

<h6>		 Defines HTML headings th, td th: is used for table header cells.
td: is used for table data cells.
<img> Defines an image src, alt src: The source location (URL) of the image file. This works the same way as the href attribute on links.
alt: The alternate text. This is used to describe the image for someone who cannot see it because they are either using a screen reader or the image src is missing. This text will only show if the image does not.
<li> Defines a list item ol, dl ol: Defines an ordered list.
dl: Defines a description list.
<table> Defines a table tr, td tr: element defines a table row.
td: element defines a table cell
<tbody> Groups the body content in a table align, valign align: Set the alignment of the content.
valign: Set the vertical alignment of the content.
<template> Defines a container for content that should be hidden when the page loads template template: is used to store the HTML code fragments, which can be cloned and inserted in an HTML document.
<kdb> Represents user input in a keyboard input or text entry field. style Allows inline styling to be applied directly to the element.
<var> Used to represent the name of a variable in a mathematical expression or programming context. n/a Represents only the name of a variable.
<ul> Used to create an unordered list. style Specifies inline CSS styles for the element.
<code> Used to define a piece of computer code. It is typically used to display code snippets within the text content of a document. n/a The 'code' element doesn't have specific attributes of its own. However, it can be combined with other elements or attributes for additional context or styling.
<ins> Used to represent text that has been inserted into the document. datetime Provides a machine-readable date and time indicating when the text was inserted.
<bdo> Used to override the Unicode bidirectional algorithm for the content within it. dir Specifies the base direction of text.
<strong> Used to indicate strong importance or emphasis. n/a The 'strong' element does not have any specific attributes of its own. It primarily focuses on the content it wraps.
<p> A block-level element that represents a paragraph of text. class, id The class attribute is used to associate an element with a particular class defined in the CSS. The id attribute provides a unique identifier for the element on the page. It is often used for more specific styling or for JavaScript to target a particular element.
